# Heading

Headings can be used for either an entire page, or for an individual section.

-----
## Heading size
A heading's `size` is separate from its `level`. The following headings are all `h1`.

<ComponentPreview code={sizeCode} />

-----
## Heading level
A heading's level can be `h1` through `h6`. By default, the size of the heading will match its level.

<ComponentPreview code={levelCode} />

-----
## Subheadings
Headings can have subheadings. Subheadings start at size `h4` and go down to `h6`. The level of a subheading will always be 1 less than the level of the heading. The subheading of an `h6` heading will be a `p`.
