# Button

It's a button.

-----
## Variants

The Button component has 4 variants - `default`, `solid`, `text`, and `link`.

-----
## Colors

The button component has 7 colors - `default`, `primary`, `secondary`, `info`, `warning`, `error`, and `success`.  

<Callout color='info' icon>Link buttons do not support colors; they are always blue.</Callout>
