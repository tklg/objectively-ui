# Button

It's a button.

-----
## Variants

The Button component has 4 variants - `default`, `solid`, `text`, and `link`.

<ComponentPreview>
  <SpaceBetween>
    <Button>Default</Button>
    <Button variant='solid'>Solid</Button>
    <Button variant='text'>Text</Button>
    <Button variant='link'>Link</Button>
  </SpaceBetween>
</ComponentPreview>

-----
## Colors

The button component has 6 colors - `default`, `primary`, `info`, `warning`, `error`, and `success`.  
Link buttons do not support colors; they are always blue.
